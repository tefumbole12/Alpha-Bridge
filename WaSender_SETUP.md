
# WaSender API Integration Setup

This document outlines how to configure the WaSender API for WhatsApp OTP delivery and notifications.

## 1. Credentials
You need two key pieces of information from your WaSender dashboard:
1. **API Key**: Found in your Profile or API Settings.
2. **Device ID (Phone Number)**: The sender identifier associated with your linked WhatsApp account.
